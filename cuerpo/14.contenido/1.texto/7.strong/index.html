La programación es un arte el cual consiste en escribir instrucciones que el computador realizará, generalmente dichas instrucciones en conjunto son consideradas programas los cuales el computador ejecutara tal y como se le indiquen.

A dichas instrucciones se les conoce como codigo fuente el cual es escrito por un programador.

Aqui veremos un ejemplo de hola mundo en algunos lenguajes de programación:

                        
Cogido en lenguaje C

                            #include<stdio.h>
                            #include<stdlib.h>
                            
                            int main() {
                                printf("Hola mundo desde el lenguaje de programación c");
                                return EXIT_SUCCESS;
                            }
                        
                        
Cogido en lenguaje C++

                            #include<iostream>
                            #include<cstdlib>

                            using namespace std;

                            int main() {
                                cout << "Hello world!" << endl;
                                return EXIT_SUCCESS;
                            }
                        
                        
Cogido en lenguaje java

                            public class Hello {
                                public static void main(String[] args) {
                                    System.out.println("Hello world");
                                }
                            }
                        
                        
Cogido en lenguaje python

                            print("Hello world!");
                        
                        
Cogido en lenguaje rust

                            fn() {
                                println!("Hello world");
                            }
                        
                        
Cogido en lenguaje JavaScript

                            console.log("Hello world");
                        
                        
Codigo en html

                            <!DOCTYPE html>
                            <html>
                                <head>
                                    <meta charset="UTF-8">
                                    <meta name="description" content="Hola mundo con html">
                                    <meta name="keywords" content="HTML, CSS, JavaScript">
                                    <title>Hola mundo</title>
                                </head>
                                <body>
                                    <h1>Hola mundo</h1>
                                </body>
                            </html>
