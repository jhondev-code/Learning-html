14.1.11 Uso de la etiqueta mark

Inicio Contactos Ayuda

La etiqueta <mark> es usada para resltar texto relevante, por ejemplo: terminos que busca el usuario. Por defecto el texto que se encuentre dentro de la etiqueta mark estan resaltados con un texto de color amarillo.
