14.4.1 Uso de <ul>

Inicio Contactos Ayuda

Al hacer uso de la etiqueta <ul> podremos crear listas desordenadas, es decir que las listas creadas no tendran un orden.

Para poder generar una lista desordenadas debemos introducir sus elementos a travez de la etiqueta <li>.

Tecnologias web:

  • HTML
  • CSS
  • JavaScript
  • Ajax
  • JQuery
  • React.js
  • Angular
  • Vue
  • Svelte
  • Bootstrap
